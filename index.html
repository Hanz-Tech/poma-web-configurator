User guide
  1. This website will only work with a recent version of Chromium based browser (Chrome, Edge...)
  2. Remove the PO-MA from the Pocket Operator,Remove microSD card
  3. Attach the PO-MA to your PC thru USB, wait for 10 secs until the LED blink in quick succession
  4. Press Connect, and select the correct device
  5. Press READ FROM ADAPTER to load config from the PO-MA.
  6. Make your changes to the red values in each box.You can also load in a .cfg or .json file
  7. Press WRITE TO ADAPTER to send red values to the adapter, no need for reboot
  8. Pesss SAVE CONFIG TO FILE to export white values to a file
